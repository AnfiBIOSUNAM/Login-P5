## Para app-flask:

Hacer entorno virtual con:

pymysql
Flask-SQLAlchemy
pycryptodomex
flask
Cryptography
flask-cors

O en su defecto, utilizar el archivo requirements.txt que se proporciona

Run:
python -m flask run

## Para app-react:

npm install
npm start

(podría requererirse instalar react-scripts y react-cookie con:

npm install <nombre_paquete>)
